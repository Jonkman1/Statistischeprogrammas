Statistische programma’s en mogelijkheden voor de nabije toekomst

Hoofdstuk 6 Python

Python is een volledig functioneel en open, programmeertaal die vanuit de computerwereld de afgelopen jaren een gelijkwaardig alternatief is geworden voor datawetenschappelijke projecten. Python is bijzonder geschikt voor de vakgebieden Deep Learning en Machine Learning, en gebruikt ook pakketten, die eenvoudig kunnen worden geïnstalleerd. Python is een veelgebruikte taal die ook wordt gebruikt bij webontwikkeling.

Sterke punten - Krachtige, volledig functionele programmeertaal.
- Biedt de basis voor objectgeoriënteerde, gestructureerde en functionele concepten.
- Het is een volwassen programmeertaal die uitgeprobeerd is en ook aangeeft waar fouten zitten wanneer je vastloopt.
- Het werkt met een groot aantal stabiele pakketten die te gebruiken zijn in de datawetenschappen en daarbuiten.
- Het werkt met een leesbare, schone syntaxis.
- Ook dit programma ontwikkelt zich voortdurende door een grote gemeenschap van ontwikkelaars.
- Volledige beschikbaarheid van de nieuwste Deep Learning en Machine Learning methoden.
- Zeer eenvoudig te automatiseren (bijvoorbeeld via scripts of een webserver).
- Volledig integreerbaar met andere programma’s (zoals Git).
- Extreem goede ondersteuning die groot is en ook nog eens voortdurend groeit. - Visualisaties zijn aantrekkelijk en eenvoudig te maken.
- Toekomstbestendig door verdere groei van het gebruik op wetenschappelijk en commercieel gebied.

Zwakke punten - Niet alle statistische methoden zijn beschikbaar.
- Sommige ontwikkelomgevingen voor de statistiek staan nog in de kinderschoenen.
- Ook hier hoge leercurve vanwege het feit dat het een “volledige” programmeertaal is.

Licentiemodel en kosten Er zijn geen gebruikerskosten voor het gebruik van Python. In sommige speciale gebieden (bijv. text mining) worden echter niet alle pakketten vrijgegeven voor commercieel gebruik.

Conclusie Python valt in deze samenvatting op omdat het een complete programmeertaal is die geschikt is voor een breed scala aan toepassingen. In de afgelopen jaren heeft het zich ook ontwikkeld tot een serieus statistisch programma door een groot aantal specifieke pakketten, en het wordt steeds populairder. Met name Python is onmisbaar voor procedures die eerder uit het vakgebied van de informatica komen, zoals Deep Learning. De voordelen zijn ook duidelijk voor de automatisering, en in interactie met andere programma’s (die ook in Python kunnen worden geschreven). Het leren van Python vereist dat men voorbereid is op het leren van een complete programmeertaal, hoewel er veel goede tutorials en trainingen over het onderwerp beschikbaar zijn vanwege de populariteit van de taal. Een specifiek op de datawetenschap toegesneden ontwikkelomgeving, zoals Rstudio, bestaat (nog) niet.
